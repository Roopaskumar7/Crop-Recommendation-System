# 🌱 Crop Recommendation System

## 📌 Overview
This project predicts the most suitable crop to grow based on soil nutrients and environmental conditions using Machine Learning.

---

## 🚀 Features
- Predicts crop based on input parameters  
- Uses trained Machine Learning model  
- Simple and user-friendly web interface  
- Real-time prediction using Flask  

---

## 🛠️ Technologies Used
- Python  
- Flask  
- Scikit-learn  
- HTML, CSS  

---

## 📊 Input Parameters
The model takes the following inputs:
- Nitrogen (N)  
- Phosphorus (P)  
- Potassium (K)  
- Temperature  
- Humidity  
- pH value  
- Rainfall  

---

## ⚙️ How to Run the Project

1. Clone the repository

   
2. Navigate to project folder  

cd Crop-Recommendation-System


3. Install required libraries  

pip install -r requirements.txt


4. Run the application  

python app.py

## 🎯 Output
- The system predicts the most suitable crop based on given inputs
